# Overview

The Assets API exposes fixed asset related functions of the Xero Accounting application and can be used for a variety of purposes such as creating assets, retrieving asset valuations etc.

## Endpoints


The base url for all assets endpoints is [https://api.xero.com/assets.xro/1.0/](https://api.xero.com/assets.xro/1.0/)

The initial set of endpoints for the Assets API includes:

GET/POST Assets

GET/POST AssetTypes

GET Settings Settings

e.g. The URL for the Assets endpoint is [https://api.xero.com/Assets.xro/1.0/Assets](https://api.xero.com/Assets.xro/1.0/Assets)

## Permissions


All API applications have access to the Assets API.

## Requests and Responses


The Assets API supports JSON only. XML is not supported.

## Versions


The Assets API is currently version 1.0 and may be updated separately to the Core API. Please see the Assets API release notes for the latest improvements.
