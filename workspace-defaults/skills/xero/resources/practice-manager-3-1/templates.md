# Templates

| Endpoint | Description |
| --- | --- |
| GET list | Return a list of all templates |

## GET list


Return a list of all templates

### Example URL

```
https://api.xero.com/practicemanager/3.1/template.api/list
```


### Example Response

```
<Response>
  <Status>OK</Status>
  <Templates>
    <Template>
      <UUID>eac5e664-f0b6-4a11-b142-5c926fdf7016</UUID>
      <Name>Brochure</Name>
    </Template>
    <Template>
      <UUID>767f2a4d-88b6-4f8a-9423-e20cae0fe726</UUID>
      <Name>Site Design</Name>
    </Template>
  </Templates>
</Response>
```
