# Overview

Xero App Store Subscriptions allows you to manage your app subscriptions, pricing and automated payment collection out-of-the-box to save time and focus otherwise spent building your own bespoke solution.

This API allows you to retrieve specific information about a specific App Store subscription.

### Availability:

App Store subscriptions is currently available in the UK, AU and NZ.

### Permissions:

The App Store subscriptions API uses the client credentials grant type
